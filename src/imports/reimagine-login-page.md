Login Page Prompt (for Figma Make)
Design a Login page for an AI-powered interior design web application called “ReImagine.”
 The interface should be minimal, modern, and easy for non-technical users, using a warm neutral color palette (beige, light brown, soft white) and soft shadows.
Layout
Split-screen layout.
Left side
Large aesthetic interior design image (modern living room)


Overlay text:


Title: “Welcome back to ReImagine”


Subtitle: “Turn your room inspiration into reality with AI-powered design.”


Right side
 Centered login card.
Login Card Components
Title: Login to your account
Input fields:
Email address


Password


Features:
Show / hide password icon


Remember me checkbox


Forgot password link


Primary button:
 Login
Secondary option below button:
 “Don’t have an account?” → Sign Up
Divider:
 “Or continue with”
Social login buttons:
Google


Apple


User Guidance
Small instruction text above form:
 “Sign in to continue designing your room with AI.”
System Feedback
The interface must support:
Loading state
 Error messages:
Invalid email or password


Network error


Success state:
Redirect to user dashboard


Design Style
Clean minimal layout


Rounded card (soft corners)


Soft shadows


Warm neutral interior design aesthetic


Large whitespace to reduce cognitive load


Accessible typography for usability


Usability Requirements
Simple for non-technical users


Clear form validation


Mobile and desktop responsive


Compatible with modern browsers (Chrome, Edge, Safari)
